Person tracking using HOG, background subtraction, Harris corner detection, centroids and Lucas Kanade.


To run the program first make sure you have all the libraries in requirements.txt.
Then run python code with "Python main.py".


DISCLAIMER: This code has problems with groups of people but there are possible solutions for those problems which I have stated in my study. For anyone intrested contact me at frlan.anton@gmail.com and I'll gladly send it over.
